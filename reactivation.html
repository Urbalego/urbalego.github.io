Pour intégrer la plate-forme Urbalego sur votre site, il vous suffit d'apporter quelques modifications minimes au code source de votre site.

Dans les exemples de code qui suivent, il vous faut remplacer "ma-commune" par l'identifiant de votre collectivité.
Cela fait, le simple ajout de ce code dans le code source de la page du guide permettra d'intégrer le service à votre site.

<script type="text/javascript">sipadm = 'ma-commune';</script>
<script src="//mip.urbalego.fr/js/integration.js" type="text/javascript"></script>
